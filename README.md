# キオクシア(285A) 株価データアーカイブ

キオクシアホールディングス(東証プライム 285A)の株価データを GitHub Actions で毎営業日 18:05 JST(保険として20:35 JSTにも再実行)に自動収集・蓄積するリポジトリです。

## 構成

| パス | 内容 |
|---|---|
| `1分足/285A_1分足_YYYY-MM-DD.csv` | 1分足(時刻,始値,高値,安値,終値,出来高) |
| `5分足/285A_5分足_YYYY-MM-DD.csv` | 5分足(同形式) |
| `日次サマリー.csv` | 日付,始値,前場引値,後場始値,終値,高値,安値,出来高 |
| `キオクシア株価収集.ps1` | 収集スクリプト(GitHub Actions・ローカル共用) |

## 仕様

- データ源: Yahoo Finance チャートAPI(銘柄コード 285A.T)
- Yahoo の1分足は約30日前までしか遡れないため、毎営業日の収集で恒久アーカイブ化している
- スクリプトは毎回、1分足28日分(7日×4チャンク)・5分足60日分を再取得して上書きする自己修復型。数日実行されなくても欠損は自動補完される
- 前場引値 = 11:30以前の最後の1分足の終値 / 後場始値 = 12:30以降の最初の1分足の始値
- 東証取引時間: 前場 9:00〜11:30、後場 12:30〜15:30(大引け)
